# 🎮 Connect Four Game (Java Swing + AI)

This is a GUI-based Connect Four game built using Java Swing with animation and an AI opponent.

---

## 🚀 Features

- 🎨 Interactive GUI using Java Swing
- ⬇️ Smooth drop animation
- 🔴🟡 Realistic circular discs
- 🤖 AI opponent using Minimax Algorithm
- ⚡ Alpha-Beta Pruning for optimization
- 👥 Two modes:
  - Player vs Player
  - Player vs Computer

---

## 🧠 AI Logic

The AI uses:
- Minimax Algorithm
- Alpha-Beta Pruning

It evaluates future moves and selects the best possible move, making it very difficult to beat.

---

## 🛠 Tech Stack

- Java
- Swing (GUI)
- Algorithms (Minimax)
